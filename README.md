# 🚨 Smart Emergency Volunteer Coordination System

An AI-powered disaster response system that intelligently assigns volunteers to tasks based on skill, location, availability, and priority.

---

## 🎯 Features

- ✅ Task Management (Add / View / Status)
- ✅ Volunteer Management (Validation included)
- ✅ AI Matching Engine (Score + Reason)
- ✅ Emergency Mode (Priority Boost + Alerts)
- ✅ Real-time Dashboard
- ✅ Animated UI with modern design

---

## 🧠 AI Logic

The system selects the best volunteer using:

- Skill Match → +50
- Same Location → +20
- Availability → +20
- Rating → +value
- Emergency Mode → +30 bonus
